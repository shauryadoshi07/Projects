📈 52-Week Stock Analysis
This project focuses on analyzing the 52-week performance of various stocks to identify investment opportunities and trends. Using Python, we collected and processed stock market data to generate key insights such as average returns, risk indicators, and patterns across industries. The analysis is aimed at both beginner and intermediate investors looking to understand how stocks behave over a full trading year.

🔍 Project Overview
Objective: Analyze the price behavior of multiple stocks over a 52-week period to evaluate performance metrics.

Data: Public stock data retrieved from financial APIs and web-scraping tools.
Tools Used: Python, Pandas, NumPy, Matplotlib, Seaborn, yfinance

📊 Key Features
Fetches historical stock data using yfinance

Computes:
52-week high and low
Average return
Volatility and risk assessment
Comparative analysis across multiple stocks
Visualizes trends and fluctuations using clear, informative charts
